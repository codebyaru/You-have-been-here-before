## Audio Format Recommendations

- **Music**: .ogg Vorbis (44.1kHz, 128kbps) - Compressed, looping
- **SFX**: .wav (44.1kHz, 16-bit) - Uncompressed, instant playback
- **Voice**: .ogg Vorbis (44.1kHz, 128kbps) - Compressed

## Free Audio Resources

- **Music**: [Incompetech](https://incompetech.com/), [OpenGameArt](https://opengameart.org/)
- **SFX**: [Freesound.org](https://freesound.org/), [Zapsplat](https://www.zapsplat.com/)
- **Voice AI**: [ElevenLabs](https://elevenlabs.io/), [Uberduck](https://uberduck.ai/)

## Notes

- All audio paths in timeline files are placeholders
- The game will function without audio files (paths just won't play anything)
- Replace placeholder files with your actual audio assets
- Maintain the same folder structure and file names
